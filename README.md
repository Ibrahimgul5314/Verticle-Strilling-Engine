# Vertical Stirling Engine – PTC Creo Parametric CAD Model

## Project Overview
3D CAD model of a Vertical Stirling Engine designed using 
PTC Creo Parametric. The model demonstrates thermodynamic 
principles, mechanical component design, and engineering 
visualization of a heat-based external combustion engine.

## Skills Demonstrated
- 3D Solid Modeling
- Mechanical Component Design
- Thermodynamic System Understanding
- Engineering Assembly Design

## Tools Used
- PTC Creo Parametric
